## ✨ Öne Çıkan Özellikler

| Özellik | Açıklama |
|---------|----------|
| 🎨 Renk Kişiselleştirme | Her not için özel renk seçeneği |
| 📌 Çoklu Not Desteği | Aynı anda birden fazla not yönetimi |
| 🔄 Otomatik Kayıt | Verileriniz otomatik olarak kaydedilir |
